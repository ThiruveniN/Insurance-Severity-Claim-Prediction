# Insurance Severity Claim Prediction

## Overview

This project develops a machine learning model to predict insurance claim severity using demographic and health-related factors.

## Objectives

- Analyze factors influencing claim costs
- Build a predictive regression model
- Evaluate model performance using RMSE and R²
- Demonstrate insurance analytics and risk assessment techniques

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Features

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Regression modelling
- Performance evaluation
- Risk driver identification

## Dataset Variables

- Age
- BMI
- Smoking Status
- Insurance Charges

## Skills Demonstrated

- Predictive Modelling
- Insurance Analytics
- Statistical Analysis
- Python Programming
- Risk Assessment
